## 수집 워크플로우

```
1. 검색어 입력 → 최신순 정렬 (기간 필터: 2024.07 ~ 2025.03)
2. 상위 200개 게시물 스캔 → 필터 조건 적용
3. 적합 이미지 선별 → 메타데이터 기록 (URL, 게시일, 계정명, 플랫폼)
4. 카테고리별 배분 확인 (볼캡/버킷햇 비율, 스타일 다양성)
5. 30장 최종 선별 → 보드(폴더) 저장
6. 출처 목록 정리 → 레퍼런스 시트 작성
```

## 실제 실행 시 도구 및 방법

> 직접 크롤링(HTTP 요청)은 SNS 약관상 제한이 있으므로 아래 방식을 권장합니다.

| 방법 | 도구 | 비고 |
|------|------|------|
| **수동 수집** | Instagram/Pinterest 앱 → 저장 | 가장 안전, 시간 소요 |
| **Pinterest 보드** | Pinterest 계정 → 보드 생성 후 핀 저장 | 협업 공유 용이 |
| **스크린샷 + 메모** | Notion / Google Slides 정리 | 팀 공유 최적 |
| **공식 API** | Instagram Graph API (비즈니스 계정) | 승인된 계정 필요 |
