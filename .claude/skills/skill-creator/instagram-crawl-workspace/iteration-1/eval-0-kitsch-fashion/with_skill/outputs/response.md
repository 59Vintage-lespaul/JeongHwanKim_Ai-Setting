# 인스타그램 키치 패션 레퍼런스 이미지 수집 — 시뮬레이션 결과

> **요청**: 인스타그램에서 키치 패션 레퍼런스 이미지 20장 수집해줘. 와키윌리 무드에 맞는 스트릿 감성으로.
> **스킬**: `instagram-crawl`
> **실행 모드**: 시뮬레이션 (실제 HTTP 요청 없음)

---

## Step 1: 해시태그 클러스터 설계

요청 분석:
- **키워드**: 키치 패션, 와키윌리 무드, 스트릿 감성
- **수량**: 20장
- **방향**: Kitsch Street & IP Universe 컨셉 — 캐릭터 그래픽, 비비드 컬러, 스트릿 스타일

브랜드 프리셋(`visual-identity.json`, `brand.config.json`) 참조 결과:
- 와키윌리 컬러 시스템: 비비드 팝 계열 (Wacky Yellow, Wacky Pink, Wacky Blue)
- 그래픽 스타일: IP 캐릭터 + 스트릿 타이포
- 코어 타겟 18~25세 자유로운 트렌드리더의 SNS 활동 태그 집중

### 설계된 해시태그 클러스터

| 레이어 | 태그 | 볼륨 추정 | 선정 이유 |
|--------|------|----------|----------|
| 핵심 태그 | `#키치패션` | ~30만건 | 메인 키워드, 최적 볼륨 |
| 핵심 태그 | `#키치코디` | ~15만건 | 코디 컷 중심 — 전신 착용샷 풍부 |
| 스타일 태그 | `#스트릿코디` | ~45만건 | 스트릿 무드 수식 |
| 스타일 태그 | `#그래픽티코디` | ~12만건 | IP·캐릭터 그래픽 티 코디 |
| 맥락 태그 | `#오오티디` | 병행 (보조) | 데일리 착장 컨텍스트 |

제외 판단:
- `#패션` — 200만+ 포화, 노이즈 과다
- `#ootd` 단독 — 관련성 낮은 게시물 범람
- 브랜드명 태그 — 홍보 게시물 편향

## Step 3: 크롤러 실행 커맨드

사용자 승인 후 실행될 bash 커맨드:

```bash
# 태그 1: 키치패션
python3 scripts/instagram-crawler/crawler.py \
  --hashtag "키치패션" \
  --count 20 \
  --no-videos 2>&1

sleep 10

# 태그 2: 캐릭터패션
python3 scripts/instagram-crawler/crawler.py \
  --hashtag "캐릭터패션" \
  --count 20 \
  --no-videos 2>&1

sleep 10

# 태그 3: 스트릿코디
python3 scripts/instagram-crawler/crawler.py \
  --hashtag "스트릿코디" \
  --count 20 \
  --no-videos 2>&1

sleep 10

# 태그 4: 그래픽티코디
python3 scripts/instagram-crawler/crawler.py \
  --hashtag "그래픽티코디" \
  --count 20 \
  --no-videos 2>&1
```

크롤러 옵션:
- `--no-videos`: Reels·동영상 제외, 이미지 포스트만
- 태그 간 `sleep 10`: Instagram 차단 방지 10초 쿨다운
- 타임아웃: 120초/태그
- 내장 품질 필터: 10KB 미만 제외, 300px 미만 해상도 제외
