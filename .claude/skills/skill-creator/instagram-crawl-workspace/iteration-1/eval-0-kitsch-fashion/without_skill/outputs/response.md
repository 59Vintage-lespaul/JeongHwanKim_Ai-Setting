# 인스타그램 키치 패션 레퍼런스 이미지 수집 계획

## 요청 해석

와키윌리 무드에 맞는 키치 스트릿 패션 레퍼런스 이미지 20장 수집 요청.
브랜드 컨셉(Kitsch Street & IP Universe)과 코어 타겟(18~25세 트렌드리더)을 기준으로 검색 전략을 수립합니다.

## 수집 도구 및 접근 방법

### 실제 사용 가능한 도구 (우선순위 순)

1. **Instagram 공식 앱/웹 수동 수집 (추천)**
   - 검색탭에서 위 해시태그 직접 검색
   - "최근" 탭보다 "인기" 탭 우선 — 이미 검증된 레퍼런스
   - 스크린샷 또는 저장 기능 활용
   - 비용 없음, 저작권 리스크 최소화

2. **Pinterest (보조 도구)**
   - 검색어: `kitsch street fashion`, `colorful kpop street style`, `harajuku 2025`
   - 큐레이션된 고퀄리티 이미지 다수
   - 무드보드 직접 생성 가능

3. **Google Images (빠른 보완)**
   - 검색어: `키치 스트릿 패션 2025`, `kitsch fashion korea street`
   - 도구 > 사용권 > 재사용 가능 필터 적용

4. **Apify Instagram Scraper (자동화 필요 시)**
   - Apify 플랫폼의 `instagram-scraper` 액터 활용
   - 해시태그 기반 최근 게시물 크롤링
   - 사용 방법:
     ```
     Input: { "hashtags": ["kitschfashion", "키치패션", "streetstyle"] }
     Output: 게시물 URL, 이미지 URL, 좋아요 수, 계정명
     ```
   - **주의**: Instagram API 정책상 상업적 무단 크롤링 제한 있음

## 이미지 선정 기준 (와키윌리 필터)

수집 후 아래 기준으로 20장 선별:

| 기준 | 세부 내용 |
|------|----------|
| 컬러 팔레트 | 비비드/네온 컬러, 멀티컬러 조합 — 무채색 단독 제외 |
| 실루엣 | 오버사이즈, 루즈핏 — 타이트 미니멀은 제외 |
| 그래픽 요소 | 캐릭터 프린트, 그래피티, 패치, 자수 등 포함 우선 |
| 스타일링 | 레이어링, 믹스매치 — 브랜드 단품 모노톤 착장 제외 |
| 연령대 무드 | 18~25세 타겟 감성 — 성숙한 하이패션 제외 |
| 시즌 적합성 | 2025 F/W 또는 현재 트렌드 반영 게시물 |

### 제외 브랜드 (레퍼런스 수집 금지)
- 리코리아
- 커버낫

## 수집 워크플로우 (단계별)

```
Step 1. 해시태그 리스트로 인스타그램 검색 (1차: 5개 태그 × 각 10장 후보 = 50장 수집)
   ↓
Step 2. 와키윌리 필터 기준 적용 → 20장으로 압축
   ↓
Step 3. 이미지 저장 위치: output/26SS/season-strategy/ 또는 해당 프로젝트 폴더
   ↓
Step 4. 파일명 규칙 적용: ref_kitsch-street-01.jpg ~ ref_kitsch-street-20.jpg
   ↓
Step 5. 무드보드 스킬(moodboard) 연결하여 시각화 가능
```

## 실행 제약 및 주의사항

- 실제 HTTP 요청/크롤링 스크립트 실행은 이 응답에서 수행하지 않음
- Instagram 공식 정책상 무단 자동 수집(scraping)은 이용약관 위반 가능성 있음
- 브랜드 내부 무드보드/레퍼런스 용도로만 사용 권장 (외부 배포 금지)
- 이미지 저작권은 원 게시자에게 있으므로 크레딧 기록 병행 권장
