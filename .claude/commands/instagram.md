# 인스타그램 이미지 수집

Instagram에서 해시태그 기반으로 이미지를 수집하고 태그별 폴더로 자동 분류합니다.

## 사용법
`/instagram [해시태그 또는 주제]` — 해시태그를 생략하면 무엇을 수집할지 물어봅니다.

## 절차

1. `skills/creative/instagram-crawl.md` 스킬 파일을 읽어 실행 절차 확인
2. 사용자 입력을 분석하여 해시태그 클러스터 설계
3. **설계한 해시태그 클러스터를 사용자에게 확인** (확인 후 진행)
4. `scripts/instagram-crawler/crawler.py`를 `--no-videos` 모드로 실행
5. 수집된 이미지를 해시태그별 폴더로 자동 분류
6. 결과 보고

## 해시태그 설계 원칙
- 구조: `핵심 태그 + 스타일 태그 + 맥락 태그`
- 한국어 태그 메인 + 영문 태그 보조 병행
- 최적 볼륨: 게시물 1만~50만 건 범위의 태그
- 수량 미지정 시 기본 20장
- 태그 간 10초 쿨다운 필수

## 예시
```
/instagram 키치패션          → 해시태그 클러스터 확인 → 20장 수집
/instagram 오버사이즈 후드 30 → 해시태그 확인 → 30장 수집
/instagram #스트릿코디        → 해시태그 확인 → 20장 수집
```

## 추가 기능
- 수집 후 "필터링해줘"라고 하면 Claude Vision AI 브랜드 적합도 필터 실행
