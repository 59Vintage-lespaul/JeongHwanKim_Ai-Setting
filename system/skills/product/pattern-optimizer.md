# 패턴 옵티마이저 (Pattern Optimizer)

> 의류 패턴 구성 분석 → 원단 소비량 계산 → 마카 효율 최적화 → 대체 소재 제안으로
> 생산 원가를 절감하고 지속가능성을 강화하는 소재 VE 심화 스킬.
> `costing-ve`의 소재 최적화 단계를 전문적으로 깊이 파고드는 확장 스킬이다.

## 기존 스킬과의 관계

| 연계 스킬 | 관계 |
|----------|------|
| `design-spec` | 아이템 스펙(패턴 피스, 소재 사양)을 입력으로 수신 |
| `costing-ve` | 소재 VE 분석 결과를 원가 계산에 연결 |
| `techpack` | 최적화된 소재 사양을 테크팩 소재 섹션에 반영 |
| `qr-process` | 절감 원가 기반 QR 리오더 의사결정 지원 |

## 언제 사용

- design-spec 완료 후, costing-ve 전 또는 병행 시
- "이 아이템 원가 너무 높아" → 소재 VE 심화가 필요할 때
- 지속가능성 이니셔티브로 친환경 소재 대안 탐색 시
- 씨빅 패딩 등 대량 생산 히어로 아이템의 마카 효율 최적화 시
- 신규 카테고리(슈즈 등) 원가 구조 파악 초기 단계

---

## 실행 절차

### Step 1: 패턴 피스 분석

아이템의 패턴 구성 요소와 각 피스의 원단 소비를 파악한다.

**표준 패턴 피스 참조:**
| 아이템 | 기본 피스 수 | 주요 구성 |
|--------|-----------|---------|
| 반팔티 | 5피스 | 앞몸판, 뒤몸판, 소매(×2), 넥밴드 |
| 후드 | 8피스+ | 앞몸판, 뒤몸판, 소매(×2), 포켓, 후드패널(×2) |
| 패딩 (씨빅) | 12피스+ | 앞/뒤 몸판, 소매, 바플 패턴, 인솔레이션 레이어 |
| 데님 팬츠 | 6피스+ | 앞판(×2), 뒤판(×2), 허리밴드, 포켓 |
| 자켓 (바시티) | 15피스+ | 몸판, 소매, 칼라, 라이닝, 리브(소매/밑단) |

**분석 결과 형식:**
```
패턴 피스 목록 — [아이템명]
총 피스 수: [N개]

| # | 피스명 | 소재 | 가로(cm) | 세로(cm) | 면적(㎡) | 비고 |
|---|--------|------|---------|---------|---------|------|
| 1 | | | | | | |
```

---

### Step 2: 원단 소비량 계산

**기본 계산:**
```
이론 소비량 = 모든 피스 면적 합계 × 패턴 배치 계수
실제 소비량 = 이론 소비량 × (1 + 로스율)
```

**로스율 기준 (아이템 유형별):**
| 아이템 유형 | 기본 로스율 | 패턴 복잡도 가산 |
|-----------|-----------|--------------|
| 티셔츠/맨투맨 | 15~20% | 그래픽 방향 맞춤 +5% |
| 아우터 (자켓/패딩) | 20~25% | 바플/절개선 복잡 +5~10% |
| 하의 (데님/팬츠) | 12~18% | 워싱 처리 +3% |
| 니트 | 10~15% | (편직 특성 별도 계산) |

**마카 효율 지표:**
```
마카 효율 = 실제 사용 면적 / 원단 총 면적 × 100
목표: 85% 이상 (우수: 88%+, 개선 필요: 80% 미만)
```

---

### Step 3: 대체 소재 매트릭스

현재 소재의 대안을 4축 평가로 비교 제시한다.

**평가 4축:**
| 축 | 기준 | 가중치 |
|---|------|--------|
| 원가 | 현재 소재 대비 절감률 | 40% |
| 품질 | 터치·내구성·세탁 견뢰도 유지 | 35% |
| 가용성 | 국내/해외 공급망 조달 용이성 | 15% |
| 지속가능성 | 재활용 소재, 인증(GRS/GOTS), 탄소발자국 | 10% |

**소재 비교표 형식:**
| 소재명 | 원가 절감 | 품질 | 가용성 | 지속가능성 | 종합점수 | 추천 여부 |
|--------|---------|------|--------|----------|---------|---------|
| 현재 소재 | 0% (기준) | | | | | — |
| 대안 A | | | | | | |
| 대안 B | | | | | | |

**커버낫 소재 원칙:**
- AUTHENTIC_LINE: 고급 소재 감도 유지 필수 — 비용 절감이 품질 타협으로 이어지면 안 됨
- COVERNAT_LINE: 코튼 중심 기본 소재 + 시즌 트렌드 소재 믹스
- 씨빅 패딩: 충전재(다운/신소재) 로스율과 충전량 정밀 관리

---

### Step 4: 원가 절감 시나리오 시뮬레이션

2~3가지 시나리오로 절감 효과를 모델링한다.

**시나리오 유형:**
| 시나리오 | 내용 |
|---------|------|
| 보수적 | 소재 대체 없이 마카 효율만 개선 |
| 표준 | 부분 소재 대체 + 마카 최적화 |
| 적극적 | 전면 소재 재검토 + 패턴 피스 최소화 |

**시뮬레이션 출력 형식:**
```
[시나리오명]
- 원단 소비량 변화: [기존 Y야드] → [개선 Y야드] ([절감%])
- 단가 변화: [기존 W원] → [개선 W원] ([절감 원/PCS])
- 1000pcs 기준 절감액: [X원]
- 시즌 총 생산량 기준 절감액: [X원]
- 마진율 영향: +[X]%p
```

---

### Step 5: 지속가능성 분석 (옵션)

친환경 소재 전환 또는 폐기물 감소 방안을 제시한다.

**분석 항목:**
- 원단 로스 감소 방안 (마카 효율 개선 → 원단 폐기물 감소)
- 재활용 소재 적용 가능성 (GRS 인증 리사이클 폴리에스터 등)
- 탄소발자국 비교 (소재별 CO₂e/kg)
- 지속가능성 인증 요건 (해외 채널 확장 대비)
