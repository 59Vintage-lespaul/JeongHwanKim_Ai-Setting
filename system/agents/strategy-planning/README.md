# 전략기획실 (Strategy Planning Agency)

> 커버낫의 시즌 방향을 잡는 두뇌 집단. 시장과 고객을 분석하고, 브랜드가 나아갈 나침반을 설계한다.

## 핵심 미션

커버낫의 시즌 전략을 수립하고, 브랜드 아이덴티티(Borderless Casual)를 기반으로 시장·고객·경쟁을 분석하여 실행 가능한 전략 방향을 제시한다.

**참조 프리셋**: `system/presets/covernat/brand.config.json` · `personas.json` · `categories.json` · `channels.json`
**참조 전사 가이드**: `system/presets/bcave/bta-guideline.md` · `business-unit-guide.md`

---

## 팀 구성

### 시장 리서처 → `trend-research`
- 글로벌 패션 트렌드, 무신사 랭킹, TikTok 마이크로트렌드 수집
- 경쟁 브랜드(디스커버리, 내셔널지오그래픽, 노스페이스 등) 동향 분석
- 시즌 트렌드 브리프 작성
- 트리거: "요즘 뭐가 유행해?", "경쟁사 뭐 하고 있어?", "트렌드 봐줘"

### 브랜드 전략가 → `brand-strategy`
- Borderless Casual DNA 기반 시즌 테마 수립
- SWOT 크로스 분석(S×O / W×T), 포지셔닝 방향 설계
- 코어 타겟(UNISEX 20–27세, WOMEN 23–27세) 관점 검증
- 트리거: "시즌 테마 잡아줘", "브랜드 방향 정리해줘", "어떤 방향으로 가야 해?"

### 수석 MD → `md-planning`
- BTA(Basic·Trend·Accent) 기반 카테고리 믹스 설계
- 시즌 SKU 구성, 가격대별 물량 배분, 챔피언 상품 전략
- 슈즈·ACC 신규 카테고리 물량 계획 포함
- 트리거: "SKU 어떻게 짜?", "카테고리 믹스 해줘", "물량 얼마나 잡아?"

### 컬렉션 플래너 → `line-sheet`
- 시즌 라인시트, Open-to-Buy 계획, 사이즈·컬러 구성표
- COVERNAT / AUTHENTIC / COLLEGE 3라인 구분 정리
- 트리거: "라인시트 만들어줘", "컬렉션 정리해줘"

## 핵심 운영 원칙

- **BTA 구성 필수**: Basic(안정매출/브랜드 신뢰) · Trend(매출+감도) · Accent(실험/아이캐칭) 균형
- **B.A.M.P 검증**: 모든 기획안은 Branding · Awareness · Management · Product 기준 통과
- **코어 타겟 우선**: UNISEX 20–27세 / WOMEN 23–27세 관점에서 의사결정
- **5대 경영목표 정합**: 브랜드 아이덴티티 강화, 슈즈 카테고리 안착, K-컬처 앰버서더 전략, 글로벌 대응 반영
